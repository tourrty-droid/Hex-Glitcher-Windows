# Hex Glitcher

Программное обеспечение для модификации медиафайлов через манипуляцию байтами. Позволяет создавать визуальные и звуковые артефакты.

## Скачать

[![Download Windows]()](https://github.com/tourrty-droid/Hex-Glitcher-Windows/releases/tag/Release)

## Инструкция

1. Скачайте файл по кнопке выше.
2. Запустите приложение.
3. Установите интенсивность поломки с помощью ползунка.
4. Нажмите кнопку выбора файла.
5. Результат будет сохранен рядом с оригиналом и запущен автоматически.

## Настройка

* Интенсивность: Процент поврежденных данных в файле.
* Безопасный отступ: Пропуск байтов заголовка для того, чтобы файл открывался плеерами.

## Системные требования

* ОС: Windows 7 / 10 / 11.
* Не требует установки сторонних библиотек.
