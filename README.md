# Hex Glitcher

Программное обеспечение для модификации медиафайлов через манипуляцию байтами. Позволяет создавать визуальные и звуковые артефакты.

## Почему Windows блокирует запуск? (ОБЯЗАТЕЛЬНО К ПРОЧТЕНИЮ!)

При запуске .exe файла может появиться синее окно "Система Windows защитила ваш компьютер".
Это происходит из-за отсутствия цифровой подписи издателя.
Как запустить:

В окне SmartScreen нажмите ссылку «Подробнее» (More info).
Нажмите кнопку «Выполнить в любом случае» (Run anyway).

Программа не является вирусом, она не вносит изменений в систему и работает только с теми файлами, которые вы выбираете сами. Весь исходный код открыт и доступен для проверки в этом репозитории.

## Инструкция

1. Скачайте файл по кнопке выше.
2. Запустите приложение.
3. Установите интенсивность поломки с помощью ползунка.
4. Нажмите кнопку выбора файла.
5. Результат будет сохранен рядом с оригиналом и запущен автоматически.

## Почему VirusTotal показывает предупреждения?

При проверке .exe файла некоторые антивирусы могут выдавать предупреждения (обычно это PyInstaller.Riskware или Generic Trojan). 

<img width="1900" height="556" alt="image" src="https://github.com/user-attachments/assets/4c8879d2-b569-46f5-949b-f25450c57842" />

1. **Это ложные срабатывания**: Они связаны с тем, что программа упакована стандартным инструментом PyInstaller и не имеет платной цифровой подписи.
2. **Открытый код**: Вы всегда можете проверить исходный код в этом репозитории и убедиться, что программа не делает ничего, кроме модификации выбранных вами файлов.
3. **Запуск из кода**: Если вы не доверяете EXE-файлу, вы можете запустить программу напрямую через Python (python main.py), тогда антивирусы не будут ругаться.
